# Python 3 Mundo 01 - Curso em Vídeo

Este repositório contém exercícios, desafios e notas sobre meus estudos de Python.

## 🛠 Requisitos

Certifique-se de que o **Python 3** está instalado.

Verifique a versão instalada:

```bash
python --version
```

ou

```bash
python3 --version
```

---

## ▶ Executando os projetos

Clone o repositório:

```bash
git clone https://github.com/ocarlosbarros/Python3Mundo01.git
```

Navegue até o diretório:

```bash
cd src
```

Execute um arquivo Python:

```bash
python filename.py
```

Exemplo:

```bash
python exercises/hello_world.py
```
